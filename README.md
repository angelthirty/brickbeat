# BRICKBEAT

키보드, 마우스, 터치로 즐기는 네온 아케이드 벽돌 깨기 게임입니다.

## 게임하기

GitHub Pages 배포가 완료되면 아래 주소에서 플레이할 수 있습니다.

https://angelthirty.github.io/brickbeat/

## 조작법

- 이동: `←` `→` 또는 `A` `D`
- 일시정지: `Space` 또는 `P`
- 마우스와 터치 조작 지원

## 버전 관리

`main` 브랜치에 변경 사항이 올라올 때마다 GitHub Actions가 자동으로 게임을 배포합니다.

## 프로젝트 구조

- `app/`: 원본 React 게임 코드
- `pages/`: GitHub Pages용 정적 게임
- `.github/workflows/`: 자동 배포 설정
